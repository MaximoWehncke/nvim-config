# Neovim Config

This is my personal neovim configuration. It's the IDE I use every single day.

## Installation 

1. Install [Neovim](https://github.com/neovim/neovim/). I've personally installed it with HomeBrew.

2. Clone this repository. Put it on `~/.config/nvim`

3. Run `nvim` and enjoy!

- Feel free to fork this repository and make your own changes.
- I've tried to keep this as simple as possible.
